# Banking Web Application

A Banking Web Application developed using ASP.NET Core and C#.

This project was created as a college practical assignment during my 6th semester to learn ASP.NET Core web development concepts such as authentication, database operations, role management, and transaction handling.

## Features

### Customer Module
- Customer Registration and Login
- View Account Details and Available Balance
- View Transaction History
- Transfer Amount to Other Accounts
- Update Customer Profile

### Staff Module
- Staff Registration and Login
- Search Customer by Account Number
- View Branch-wise Customer and Employee Details
- Update Staff Profile

## Technologies Used
- ASP.NET Core
- C# 
- MVC Framework / SQL Server
- HTML, CSS, Bootstrap
- Visual Studio

## Purpose
This project was developed for learning and practice purposes as part of academic coursework.
